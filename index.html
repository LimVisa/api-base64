API Usage:

Make a GET request to the endpoint with a Base64 code parameter:

GET /api/base64?code=[your_base64_string_here]

The API will return the image directly, which can be displayed in img tags:

<img src="/api/base64?code=[base64_string]" alt="Converted image">
